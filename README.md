# 🌸 LARAJEAN – Main Character Aesthetic Website

A soft, clean-girl, romcom-inspired website with an AI Bestie chatbot and mood-based experiences.

## ✨ Features

### 💗 Mood Selector (Affirmations)
- Choose your vibe: Lover, Reputation, Serene
- Get 3 random main-character affirmations
- Flip card animation
- Soft pastel UI

### 🎧 Music Era Page
- Taylor Swift era inspired colors
- Lover / Reputation / Serene themes
- One demo song with play/pause

### 🤖 Bestie Chatbot (In Progress → ML)
- Cute chat UI with typing animation
- Mood-based replies (happy, sad, tired, angry)
- Will be upgraded to:
  - Emotion detection using Machine Learning
  - Personalized responses
  - Mood-based music & affirmation recommendation

## 🧠 Future ML Plan

This project will be converted into an:
**Emotion-Aware Virtual Best Friend using NLP & Machine Learning**

Pipeline:
1. User types how they feel
2. ML model predicts emotion (Sentiment Analysis)
3. Chatbot responds accordingly
4. UI theme + music + affirmations adapt
